# Cartesia API — Overview

## O que é
API de Text-to-Speech (TTS) de alta qualidade para gerar áudio a partir de texto.
Modelo atual: **sonic-2**.

## Base URL
```
https://api.cartesia.ai
```

## Autenticação
```
X-API-Key: {CARTESIA_API_KEY}
Cartesia-Version: 2024-06-10
Content-Type: application/json
```

## Casos de Uso na Piranha Global
- Gerar áudios de voz para envio via WhatsApp
- Gravar mensagens de voz personalizadas
- Áudio para ligações automatizadas

## Documentação Oficial
https://docs.cartesia.ai
