# Task: Criar Novo Script de Automação

## Objetivo
Criar um script Python de automação que integre com as APIs da Piranha Global.

## Workflow
1. `@analyst *gather-requirements` — levantar requisitos
2. `@architect *design-solution` — desenhar arquitetura
3. `@dev *write-code` — implementar
4. `@qa *review-build` — revisar qualidade
5. Human approval — você aprova antes de fazer deploy

## Input Necessário
- Descrição do que o script deve fazer
- APIs envolvidas (shopify/evolution/ultravox/cartesia/telnyx)
- Frequência de execução (manual/agendado)
- Dados de entrada e saída esperados

## Exemplo de Ativação
```
@analyst Preciso de um script que consulte pedidos abandonados na Shopify e envie
uma mensagem de recuperação via WhatsApp usando a Evolution API.
O script deve rodar a cada 2 horas.
```
